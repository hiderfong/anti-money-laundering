## 一、本轮落地项

### 请求追踪 ID

新增 `RequestTraceFilter`：

| 项目 | 行为 |
|------|------|
| 请求头 | 优先读取 `X-Request-Id` |
| 无请求头 | 自动生成 32 位无横线 UUID |
| 响应头 | 回传 `X-Request-Id` |
| 日志 MDC | 写入 `traceId`，匹配现有 logback `%X{traceId}` |
| 审计日志 | `AuditLogAspect` 使用同一个 `traceId` 写入 `t_audit_log.trace_id` |

### 验收方式

```bash
curl -i http://127.0.0.1:8080/api/system/health \
  -H 'X-Request-Id: aml-trace-smoke-001'
```

期望响应头包含：

```text
X-Request-Id: aml-trace-smoke-001
```

执行带 `@AuditLog` 的写接口后，可在 `t_audit_log.trace_id` 查询同一 traceId。

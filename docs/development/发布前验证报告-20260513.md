## 阻塞项

### P0：本机 BuildKit/buildx 缺失

本机运行：

```bash
DOCKER_BUILDKIT=1 docker build -t aml-system:release-precheck-20260513 -f docker/Dockerfile .
```

失败：

```text
ERROR: BuildKit is enabled but the buildx component is missing or broken.
```

建议修复本机 Docker 工具链后重跑镜像构建：

```bash
brew install docker-buildx
```

并按 Homebrew caveat 确认 Docker CLI 能发现插件：

```json
{
  "cliPluginsExtraDirs": [
    "/opt/homebrew/lib/docker/cli-plugins"
  ]
}
```

修复后验证：

```bash
docker buildx version
DOCKER_BUILDKIT=1 docker build -t aml-system:release-precheck-20260513 -f docker/Dockerfile .
```

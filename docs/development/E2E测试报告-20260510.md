## 测试过程中的修复

在测试过程中发现并修复了以下问题：

1. **STR 报告接口路径**
   - 问题：前端调用 `/str-reports`，后端映射为 `/cases/str-reports`
   - 修复：后端改为 `/str-reports`，添加 `/str-reports/page` 分页端点
   - 提交：bdfe4dc, 8bccb59

2. **RBAC 角色映射**
   - 问题：prepare-rbac-e2e-users.sh 脚本角色 ID 与数据库不匹配
   - 修复：手动通过 Docker 执行正确的 SQL
   - 建议：更新脚本使用正确的角色 ID

3. **no-redis profile 配置**
   - 问题：`dev,no-redis` profile 未自动禁用 Redis
   - 修复：需额外添加 `--aml.redis.enabled=false` 参数
   - 建议：在 no-redis profile 中默认设置 `aml.redis.enabled=false`
